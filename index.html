ByteBeats

OSAMC 4/2016

What is a bytebeat?

a waveform calculated as a function of time

A typical bytebeat
  • has no score
  • uses integer arithmetic
  • has no concept of instruments
  • uses output sample rate and bit depth
  • is a one-liner in the language of your choice
Simple example at 8khz, 8bit

((t >> 11) % 4) * t


  • implicit % 255 because of unsigned char
  • therefore t produces a sawtooth at 8000 / 255 =~ 31Hz
  • >> 11 works as a slower counter repeating values (0, 1, 2, 3)
    it changes every 2048 samples = ~0.25s
Simple example at 8khz, 8bit

((t >> 11) & 3) * t


Sierpinsky Harmonies with bit operators

(t >> 5) & t


  • creates complex harmonies of square waves
  • in octaves, if using shift operator
  • but not when using arbitrary multiplication/division
